The **Counter Blox Script** stores all user preferences in a local config file. You can edit settings directly through the in-game GUI or modify the configuration file manually for more granular control. Below is the default config structure and a reference table explaining each parameter.

| Parameter | Default Value | Range | Description |

| `aimbot_smoothing` | `0.4` | `0.1–1.0` | Camera smoothing factor (lower = snappier) |
| `aimbot_bone` | `Head` | `Head/Torso/Random` | Target bone for **Counter Blox Script Aimbot** |
| `esp_enabled` | `true` | `true/false` | Master toggle for **ESP** overlay |
| `esp_show_health` | `true` | `true/false` | Display health bars above players |
| `esp_show_distance` | `true` | `true/false` | Show distance tags on **ESP** boxes |
| `hitbox_scale` | `2.0` | `1.0–6.0` | **Hitbox** expansion multiplier |
| `speed_multiplier` | `1.0` | `1.0–2.5` | Movement speed override |
| `anti_afk` | `true` | `true/false` | Keep session alive when idle |
| `theme` | `dark` | `dark/light/custom` | GUI color theme |

```bash
# ============================================
# Counter Blox Script — Configuration File
# Location: C:/Users/YourName/CounterBloxScript/config.lua
# ============================================

-- Aimbot Settings
Config = {
    Aimbot = {
        Enabled      = true,
        FOV          = 120,
        Smoothing    = 0.4,
        TargetBone   = "Head",       -- Options: "Head", "Torso", "Random"
        SilentAim    = true,
        TeamCheck    = true,         -- Skip teammates
        VisibleCheck = true,         -- Only aim at visible targets
        Keybind      = "CapsLock"    -- Toggle key

    -- ESP Settings
    ESP = {
        Enabled       = true,

        ShowHealth    = true,
        ShowDistance   = true,

    -- Hitbox Settings
    Hitbox = {
        Enabled       = true,
        Scale         = 2.0,
        HeadScale     = 3.0,
        TorsoScale    = 2.0,
        Transparency  = 0.5

    -- Movement Settings

        SpeedEnabled  = false,
```

## 🛠️ Troubleshooting Common Issues

- ❌ **Script fails to execute after loading**
  The executor may not support the required API level. Ensure you are using a Level 7+ executor.
  **Solution:** Update your executor to its latest version, then re-inject the **Counter Blox Script**

- ❌ **GUI does not appear in-game**
  This typically happens when the executor's drawing library is outdated or incompatible.
  **Solution:** Switch to a different executor or check if a drawing library update is available for your current one

- ❌ **Aimbot locks onto teammates**
  The `TeamCheck` flag may be disabled in your configuration.
  **Solution:** Open the config file at `C:/Users/.../CounterBloxScript/config.lua` and set `TeamCheck = true`

- ❌ **ESP overlays cause frame drops**
  Rendering full skeleton wireframes on all players is GPU-intensive, especially with the **Counter Blox Script Aimbot ESP** suite active.
  **Solution:** Disable `ShowSkeleton` and reduce `MaxDistance` to `800` to lighten the rendering load

- ❌ **Hitbox scaling appears inconsistent**
  Some servers run modified collision logic that conflicts with client-side **Hitbox** changes.
  **Solution:** Lower the `Scale` value to `2.0` or below and enable `Transparency` to visually confirm the actual hitbox boundaries

- ❌ **Auto-updater fails to connect**
  Your firewall or antivirus may be blocking outbound connections to the **Counter Blox Script Github** repository.
  **Solution:** Whitelist the script's update URL in your firewall settings or update manually by downloading the latest release
